> 
> ```css
> table {

>   width: 100%;

> td, th {
>   vertical-align: top;
>   overflow-wrap: anywhere;
>   word-break: break-word;
> }
> 
> td textarea,
> td input,
> td select {
>   width: 100%;
>   max-width: 100%;
> }
> 
> textarea {
>   resize: vertical;
>   overflow: hidden;
>   min-height: 3.5rem;
>   white-space: pre-wrap;
> }
> 
> @media print {
>   textarea {
>     overflow: visible !important;
>     height: auto !important;
>     min-height: 3.5rem;
>     white-space: pre-wrap !important;
>   }
> 
>   td {
>     page-break-inside: avoid;
>     break-inside: avoid;
>   }
> }
> ```
> 
> And the children row should change from this kind of thing:
> 
> ```html
> <td><input data-field="child1_contact" type="text" /></td>
> ```
> 
> To this:
> 
> ```html
> <td>
>   <textarea data-field="child1_contact" rows="2"></textarea>
> </td>
> ```
> 
> Same for notes:
> 
> ```html
> <td>
>   <textarea data-field="child1_notes" rows="2"></textarea>

> ```
>
