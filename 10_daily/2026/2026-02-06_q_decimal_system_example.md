> An orphan is any file that fails one of these checks:

> Filename doesn’t match:

> - or allowed short form: `AA_SUBJECT_desc.ext`

> Filename says `AA_...` but it’s not inside the matching top folder.

> - file name starts with `60_...`

> A file exists outside your 10 top folders.

> File is present but not in `FILE_MANIFEST.csv` (should never happen after a run)
