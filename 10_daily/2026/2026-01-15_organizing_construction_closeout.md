> # LAYER 4 — BINDER ASSEMBLY SCRIPT (AUTOMATION)
> 
> ## Script Purpose
> - Deterministically assemble
> - No interpretation

> 
> ---

> ## Script Inputs
> - QBO ZIP export

> ## Script Outputs
> - Cover page
> - TOC (auto-generated from folder/file order)
> - Section dividers
> - Merged PDF
> 
> ---
> 
> ## Python Script (Skeleton – SAFE)
> 
> ```python
> from pathlib import Path
> from PyPDF2 import PdfMerger
> 
> ROOT = Path("AUDIT_READY_CLOSEOUT")
> OUTPUT = "Rays_Construction_Audit_Binder.pdf"
> 
> merger = PdfMerger()
> 
> for folder in sorted(ROOT.iterdir()):
>     if folder.is_dir():
>         # Add section divider (pre-made PDF)
>         divider = Path("dividers") / f"{folder.name}.pdf"
>         if divider.exists():
>             merger.append(str(divider))
>         for file in sorted(folder.glob("*.pdf")):
>             merger.append(str(file))
> 
> merger.write(OUTPUT)
> merger.close()
> ```
> 
> **Important:**  
> Dividers and cover pages are **static templates**, not generated dynamically.
