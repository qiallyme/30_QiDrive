> # Phase 1 — Turn Obsidian into a mobile site (Quartz)

> - Built for Obsidian vaults
> - Graph view + backlinks
> - Fast mobile navigation
> - Zero database required
> - Perfect for Cloudflare Pages
> 
> ### Install Quartz locally
> 
> Inside your repo folder:
> 
> ```bash
> git clone https://github.com/jackyzha0/quartz.git my-brain-site
> cd my-brain-site
> npm install
> ```

> Now connect your vault.
> 
> Put your Obsidian vault inside:
> 
> ```
> /content
> ```
>

> ```

>     LEGAL/

> 
> ```bash
> npx quartz build

> 
> ```

> ```

> - Supabase Postgres (pgvector)
> - Cloudflare Worker
> - OpenAI embeddings (or local later)

> ### Python script to embed vault
> 
> This scans your Quartz `/content` folder and sends notes to Supabase.

> Core idea:
> 
> ```python

>     chunks = split(text)
>     embedding = openai.embed(chunks)
>     store_in_supabase(embedding, metadata=file)
