# 30_QiDrive

`30_QiDrive` is the mount/junction point pointing to **MyDrive** in **Google Drive**. 

It serves as the primary storage location for large assets, archives, and binaries within the QiLabs ecosystem.

## Git Status

> [!IMPORTANT]
> **This directory is NOT tracked in `.git`.** 

Due to the nature of Google Drive storage (which includes large files, private documents, media assets, and sensitive logs), tracking these files in version control is omitted. 

If this folder appears in your local dev tree as an untracked directory or junction, it should remain ignored via `.gitignore` or bypassed during commits.
