## Article 4. Root Folder Boundaries

Stable ID: `T10.A04`

### Sec. 10.04.010. Single Domain Rule

Stable ID: `T10.A04.S010`

- `Line 1` Each root folder governs exactly one domain; cross-domain material must be placed in the owning domain. (`Sec. 10.04.010.L001`; stable ID `T10.A04.S010.L001`)
- `Line 2` Shared dependencies must be documented with source mappings in the relevant title. (`Sec. 10.04.010.L002`; stable ID `T10.A04.S010.L002`)
- `Line 3` Boundary disputes are resolved by reviewing the primary function in the domain's QiCode title. (`Sec. 10.04.010.L003`; stable ID `T10.A04.S010.L003`)
