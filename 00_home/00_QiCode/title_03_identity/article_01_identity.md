## Article 1. Identity

Stable ID: `T03.A01`

### Sec. 3.01.010. Self-Definition

Stable ID: `T03.A01.S010`

- `Line 1` I retain the right to define myself in language that is precise, alive, and not reduced to someone else's fear. (`Sec. 3.01.010.L001`; stable ID `T03.A01.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 3.01.010.L002`; stable ID `T03.A01.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 3.01.010.L003`; stable ID `T03.A01.S010.L003`)

### Sec. 3.01.020. Multiplicity

Stable ID: `T03.A01.S020`

- `Line 1` Different roles, moods, capacities, and histories may coexist without becoming fraud or contradiction. (`Sec. 3.01.020.L001`; stable ID `T03.A01.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 3.01.020.L002`; stable ID `T03.A01.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 3.01.020.L003`; stable ID `T03.A01.S020.L003`)

### Sec. 3.01.030. Naming

Stable ID: `T03.A01.S030`

- `Line 1` Names, labels, and aliases should clarify relationship and context without replacing the living person. (`Sec. 3.01.030.L001`; stable ID `T03.A01.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 3.01.030.L002`; stable ID `T03.A01.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 3.01.030.L003`; stable ID `T03.A01.S030.L003`)
