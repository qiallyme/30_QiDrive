## Article 3. Boundaries

Stable ID: `T03.A03`

### Sec. 3.03.010. Boundary Validity

Stable ID: `T03.A03.S010`

- `Line 1` A boundary is valid when it protects safety, truth, capacity, dignity, or consent. (`Sec. 3.03.010.L001`; stable ID `T03.A03.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 3.03.010.L002`; stable ID `T03.A03.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 3.03.010.L003`; stable ID `T03.A03.S010.L003`)

### Sec. 3.03.020. Access

Stable ID: `T03.A03.S020`

- `Line 1` Access to me, my labor, my attention, or my records is not automatically granted by proximity or urgency. (`Sec. 3.03.020.L001`; stable ID `T03.A03.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 3.03.020.L002`; stable ID `T03.A03.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 3.03.020.L003`; stable ID `T03.A03.S020.L003`)

### Sec. 3.03.030. Enforcement

Stable ID: `T03.A03.S030`

- `Line 1` A boundary without enforcement becomes a request; enforcement may be calm, documented, and proportionate. (`Sec. 3.03.030.L001`; stable ID `T03.A03.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 3.03.030.L002`; stable ID `T03.A03.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 3.03.030.L003`; stable ID `T03.A03.S030.L003`)
