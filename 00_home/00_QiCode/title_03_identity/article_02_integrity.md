## Article 2. Integrity

Stable ID: `T03.A02`

### Sec. 3.02.010. Alignment

Stable ID: `T03.A02.S010`

- `Line 1` Integrity means the visible life and the known inner truth are brought into closer alignment over time. (`Sec. 3.02.010.L001`; stable ID `T03.A02.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 3.02.010.L002`; stable ID `T03.A02.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 3.02.010.L003`; stable ID `T03.A02.S010.L003`)

### Sec. 3.02.020. Record

Stable ID: `T03.A02.S020`

- `Line 1` When identity is challenged or mischaracterized, records should preserve context rather than retaliatory distortion. (`Sec. 3.02.020.L001`; stable ID `T03.A02.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 3.02.020.L002`; stable ID `T03.A02.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 3.02.020.L003`; stable ID `T03.A02.S020.L003`)

### Sec. 3.02.030. Correction

Stable ID: `T03.A02.S030`

- `Line 1` Correction is lawful when it restores truth without unnecessary exposure. (`Sec. 3.02.030.L001`; stable ID `T03.A02.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 3.02.030.L002`; stable ID `T03.A02.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 3.02.030.L003`; stable ID `T03.A02.S030.L003`)
