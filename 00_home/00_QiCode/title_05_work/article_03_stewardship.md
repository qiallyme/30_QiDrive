## Article 3. Stewardship

Stable ID: `T05.A03`

### Sec. 5.03.010. Resources

Stable ID: `T05.A03.S010`

- `Line 1` Money, tools, records, systems, and energy are stewardship objects and should be treated with care. (`Sec. 5.03.010.L001`; stable ID `T05.A03.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 5.03.010.L002`; stable ID `T05.A03.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 5.03.010.L003`; stable ID `T05.A03.S010.L003`)

### Sec. 5.03.020. Sustainability

Stable ID: `T05.A03.S020`

- `Line 1` Work that destroys the worker is not a stable operating model. (`Sec. 5.03.020.L001`; stable ID `T05.A03.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 5.03.020.L002`; stable ID `T05.A03.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 5.03.020.L003`; stable ID `T05.A03.S020.L003`)

### Sec. 5.03.030. Value

Stable ID: `T05.A03.S030`

- `Line 1` Value includes money, care, time, skill, meaning, continuity, and repair. (`Sec. 5.03.030.L001`; stable ID `T05.A03.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 5.03.030.L002`; stable ID `T05.A03.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 5.03.030.L003`; stable ID `T05.A03.S030.L003`)
