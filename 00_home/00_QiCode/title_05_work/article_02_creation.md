## Article 2. Creation

Stable ID: `T05.A02`

### Sec. 5.02.010. Useful First

Stable ID: `T05.A02.S010`

- `Line 1` A created artifact should be useful before it is ornamental. (`Sec. 5.02.010.L001`; stable ID `T05.A02.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 5.02.010.L002`; stable ID `T05.A02.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 5.02.010.L003`; stable ID `T05.A02.S010.L003`)

### Sec. 5.02.020. Voice

Stable ID: `T05.A02.S020`

- `Line 1` Creative work may carry personal truth while still being edited for audience, safety, and form. (`Sec. 5.02.020.L001`; stable ID `T05.A02.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 5.02.020.L002`; stable ID `T05.A02.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 5.02.020.L003`; stable ID `T05.A02.S020.L003`)

### Sec. 5.02.030. Output

Stable ID: `T05.A02.S030`

- `Line 1` Outputs should name their audience, status, and intended next use. (`Sec. 5.02.030.L001`; stable ID `T05.A02.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 5.02.030.L002`; stable ID `T05.A02.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 5.02.030.L003`; stable ID `T05.A02.S030.L003`)
