## Article 1. Work

Stable ID: `T05.A01`

### Sec. 5.01.010. Project Home

Stable ID: `T05.A01.S010`

- `Line 1` Every serious project should have one clear home, one purpose, and one visible next-action surface. (`Sec. 5.01.010.L001`; stable ID `T05.A01.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 5.01.010.L002`; stable ID `T05.A01.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 5.01.010.L003`; stable ID `T05.A01.S010.L003`)

### Sec. 5.01.020. Labor Record

Stable ID: `T05.A01.S020`

- `Line 1` Labor that costs time, income, health, or opportunity may be recorded without shame. (`Sec. 5.01.020.L001`; stable ID `T05.A01.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 5.01.020.L002`; stable ID `T05.A01.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 5.01.020.L003`; stable ID `T05.A01.S020.L003`)

### Sec. 5.01.030. Completion

Stable ID: `T05.A01.S030`

- `Line 1` Completion means the intended user can find and use the result without reconstructing hidden context. (`Sec. 5.01.030.L001`; stable ID `T05.A01.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 5.01.030.L002`; stable ID `T05.A01.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 5.01.030.L003`; stable ID `T05.A01.S030.L003`)
