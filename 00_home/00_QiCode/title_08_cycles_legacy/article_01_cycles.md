## Article 1. Cycles

Stable ID: `T08.A01`

### Sec. 8.01.010. Seasonality

Stable ID: `T08.A01.S010`

- `Line 1` Life moves in seasons; a system should allow return, renewal, rest, and revision. (`Sec. 8.01.010.L001`; stable ID `T08.A01.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 8.01.010.L002`; stable ID `T08.A01.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 8.01.010.L003`; stable ID `T08.A01.S010.L003`)

### Sec. 8.01.020. Review

Stable ID: `T08.A01.S020`

- `Line 1` Active commitments, systems, and records need review before they become hidden burden. (`Sec. 8.01.020.L001`; stable ID `T08.A01.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 8.01.020.L002`; stable ID `T08.A01.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 8.01.020.L003`; stable ID `T08.A01.S020.L003`)

### Sec. 8.01.030. Reset

Stable ID: `T08.A01.S030`

- `Line 1` A reset is lawful when the current structure no longer serves truth, care, or action. (`Sec. 8.01.030.L001`; stable ID `T08.A01.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 8.01.030.L002`; stable ID `T08.A01.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 8.01.030.L003`; stable ID `T08.A01.S030.L003`)
