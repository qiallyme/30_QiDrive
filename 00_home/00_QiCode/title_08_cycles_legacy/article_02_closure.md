## Article 2. Closure

Stable ID: `T08.A02`

### Sec. 8.02.010. Completion Record

Stable ID: `T08.A02.S010`

- `Line 1` Closed work should say what was done, where it lives, and what remains open. (`Sec. 8.02.010.L001`; stable ID `T08.A02.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 8.02.010.L002`; stable ID `T08.A02.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 8.02.010.L003`; stable ID `T08.A02.S010.L003`)

### Sec. 8.02.020. Grief

Stable ID: `T08.A02.S020`

- `Line 1` Closure may include grief without treating grief as failure. (`Sec. 8.02.020.L001`; stable ID `T08.A02.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 8.02.020.L002`; stable ID `T08.A02.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 8.02.020.L003`; stable ID `T08.A02.S020.L003`)

### Sec. 8.02.030. Archive

Stable ID: `T08.A02.S030`

- `Line 1` Archive material should remain findable without crowding the active field. (`Sec. 8.02.030.L001`; stable ID `T08.A02.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 8.02.030.L002`; stable ID `T08.A02.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 8.02.030.L003`; stable ID `T08.A02.S030.L003`)
