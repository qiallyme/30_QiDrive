## Article 2. Law of One

Stable ID: `T01.A02`

### Sec. 1.02.010. Unity

Stable ID: `T01.A02.S010`

- `Line 1` The Law of One is held here as a unity principle: separation is useful for navigation, but not ultimate truth. (`Sec. 1.02.010.L001`; stable ID `T01.A02.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 1.02.010.L002`; stable ID `T01.A02.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 1.02.010.L003`; stable ID `T01.A02.S010.L003`)

### Sec. 1.02.020. Service

Stable ID: `T01.A02.S020`

- `Line 1` Service must not require self-abandonment; true service preserves dignity, consent, and source alignment. (`Sec. 1.02.020.L001`; stable ID `T01.A02.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 1.02.020.L002`; stable ID `T01.A02.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 1.02.020.L003`; stable ID `T01.A02.S020.L003`)

### Sec. 1.02.030. Discernment

Stable ID: `T01.A02.S030`

- `Line 1` Unity does not mean collapse of boundaries; discernment is how unity becomes safe in daily life. (`Sec. 1.02.030.L001`; stable ID `T01.A02.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 1.02.030.L002`; stable ID `T01.A02.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 1.02.030.L003`; stable ID `T01.A02.S030.L003`)
