## Article 3. Sacred Interior

Stable ID: `T02.A03`

### Sec. 2.03.010. Privacy

Stable ID: `T02.A03.S010`

- `Line 1` Not every inner truth is public material, even when it is real. (`Sec. 2.03.010.L001`; stable ID `T02.A03.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 2.03.010.L002`; stable ID `T02.A03.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 2.03.010.L003`; stable ID `T02.A03.S010.L003`)

### Sec. 2.03.020. Discernment

Stable ID: `T02.A03.S020`

- `Line 1` Inner guidance should be recorded with humility and checked against care, reality, and consequence. (`Sec. 2.03.020.L001`; stable ID `T02.A03.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 2.03.020.L002`; stable ID `T02.A03.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 2.03.020.L003`; stable ID `T02.A03.S020.L003`)

### Sec. 2.03.030. Integration

Stable ID: `T02.A03.S030`

- `Line 1` Inner work becomes useful when it changes action, language, boundaries, or peace. (`Sec. 2.03.030.L001`; stable ID `T02.A03.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 2.03.030.L002`; stable ID `T02.A03.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 2.03.030.L003`; stable ID `T02.A03.S030.L003`)
