## Article 1. Attention

Stable ID: `T04.A01`

### Sec. 4.01.010. Capture Before Sort

Stable ID: `T04.A01.S010`

- `Line 1` When attention is overloaded, capture first and sort later. (`Sec. 4.01.010.L001`; stable ID `T04.A01.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 4.01.010.L002`; stable ID `T04.A01.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 4.01.010.L003`; stable ID `T04.A01.S010.L003`)

### Sec. 4.01.020. Focus

Stable ID: `T04.A01.S020`

- `Line 1` Focus requires a named object, a realistic scope, and permission to stop. (`Sec. 4.01.020.L001`; stable ID `T04.A01.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 4.01.020.L002`; stable ID `T04.A01.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 4.01.020.L003`; stable ID `T04.A01.S020.L003`)

### Sec. 4.01.030. Noise

Stable ID: `T04.A01.S030`

- `Line 1` Noise should be reduced at the system level rather than treated only as personal discipline failure. (`Sec. 4.01.030.L001`; stable ID `T04.A01.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 4.01.030.L002`; stable ID `T04.A01.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 4.01.030.L003`; stable ID `T04.A01.S030.L003`)
