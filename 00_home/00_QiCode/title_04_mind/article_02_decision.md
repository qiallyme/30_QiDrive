## Article 2. Decision

Stable ID: `T04.A02`

### Sec. 4.02.010. Decision Record

Stable ID: `T04.A02.S010`

- `Line 1` Important decisions should preserve context, options, choice, and consequence. (`Sec. 4.02.010.L001`; stable ID `T04.A02.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 4.02.010.L002`; stable ID `T04.A02.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 4.02.010.L003`; stable ID `T04.A02.S010.L003`)

### Sec. 4.02.020. Delay

Stable ID: `T04.A02.S020`

- `Line 1` Delay may be lawful when information, safety, or capacity is missing. (`Sec. 4.02.020.L001`; stable ID `T04.A02.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 4.02.020.L002`; stable ID `T04.A02.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 4.02.020.L003`; stable ID `T04.A02.S020.L003`)

### Sec. 4.02.030. Reversibility

Stable ID: `T04.A02.S030`

- `Line 1` A decision should name whether it is reversible, difficult to reverse, or final. (`Sec. 4.02.030.L001`; stable ID `T04.A02.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 4.02.030.L002`; stable ID `T04.A02.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 4.02.030.L003`; stable ID `T04.A02.S030.L003`)
