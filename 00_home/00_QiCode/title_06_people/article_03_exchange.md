## Article 3. Exchange

Stable ID: `T06.A03`

### Sec. 6.03.010. Reciprocity

Stable ID: `T06.A03.S010`

- `Line 1` Exchange should be grounded in clarity, consent, and reality rather than guilt or panic. (`Sec. 6.03.010.L001`; stable ID `T06.A03.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 6.03.010.L002`; stable ID `T06.A03.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 6.03.010.L003`; stable ID `T06.A03.S010.L003`)

### Sec. 6.03.020. Care Labor

Stable ID: `T06.A03.S020`

- `Line 1` Care labor is real labor even when performed inside family systems. (`Sec. 6.03.020.L001`; stable ID `T06.A03.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 6.03.020.L002`; stable ID `T06.A03.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 6.03.020.L003`; stable ID `T06.A03.S020.L003`)

### Sec. 6.03.030. Obligation

Stable ID: `T06.A03.S030`

- `Line 1` Obligation should not be inferred from crisis pressure without review. (`Sec. 6.03.030.L001`; stable ID `T06.A03.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 6.03.030.L002`; stable ID `T06.A03.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 6.03.030.L003`; stable ID `T06.A03.S030.L003`)
