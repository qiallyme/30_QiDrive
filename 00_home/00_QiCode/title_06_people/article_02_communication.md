## Article 2. Communication

Stable ID: `T06.A02`

### Sec. 6.02.010. Context

Stable ID: `T06.A02.S010`

- `Line 1` Important communication should preserve sender, receiver, date, medium, and surrounding context. (`Sec. 6.02.010.L001`; stable ID `T06.A02.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 6.02.010.L002`; stable ID `T06.A02.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 6.02.010.L003`; stable ID `T06.A02.S010.L003`)

### Sec. 6.02.020. Tone and Fact

Stable ID: `T06.A02.S020`

- `Line 1` Tone may be described, but fact and interpretation should remain distinguishable. (`Sec. 6.02.020.L001`; stable ID `T06.A02.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 6.02.020.L002`; stable ID `T06.A02.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 6.02.020.L003`; stable ID `T06.A02.S020.L003`)

### Sec. 6.02.030. Draft Status

Stable ID: `T06.A02.S030`

- `Line 1` Draft replies remain drafts until deliberately sent. (`Sec. 6.02.030.L001`; stable ID `T06.A02.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 6.02.030.L002`; stable ID `T06.A02.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 6.02.030.L003`; stable ID `T06.A02.S030.L003`)
