## Article 1. Ethics

Stable ID: `T07.A01`

### Sec. 7.01.010. Dignity

Stable ID: `T07.A01.S010`

- `Line 1` Records should preserve dignity even when documenting conflict, harm, or contradiction. (`Sec. 7.01.010.L001`; stable ID `T07.A01.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 7.01.010.L002`; stable ID `T07.A01.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 7.01.010.L003`; stable ID `T07.A01.S010.L003`)

### Sec. 7.01.020. Minimum Necessary Disclosure

Stable ID: `T07.A01.S020`

- `Line 1` Share only what is needed for the purpose, audience, and safety context. (`Sec. 7.01.020.L001`; stable ID `T07.A01.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 7.01.020.L002`; stable ID `T07.A01.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 7.01.020.L003`; stable ID `T07.A01.S020.L003`)

### Sec. 7.01.030. Consent

Stable ID: `T07.A01.S030`

- `Line 1` Consent and access boundaries should be explicit when material involves other people. (`Sec. 7.01.030.L001`; stable ID `T07.A01.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 7.01.030.L002`; stable ID `T07.A01.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 7.01.030.L003`; stable ID `T07.A01.S030.L003`)
