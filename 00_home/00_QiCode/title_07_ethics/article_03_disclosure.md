## Article 3. Disclosure

Stable ID: `T07.A03`

### Sec. 7.03.010. Sensitive Material

Stable ID: `T07.A03.S010`

- `Line 1` Medical, legal, financial, and family-conflict material requires higher review before disclosure. (`Sec. 7.03.010.L001`; stable ID `T07.A03.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 7.03.010.L002`; stable ID `T07.A03.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 7.03.010.L003`; stable ID `T07.A03.S010.L003`)

### Sec. 7.03.020. Audience

Stable ID: `T07.A03.S020`

- `Line 1` A record may be true and still not appropriate for every audience. (`Sec. 7.03.020.L001`; stable ID `T07.A03.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 7.03.020.L002`; stable ID `T07.A03.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 7.03.020.L003`; stable ID `T07.A03.S020.L003`)

### Sec. 7.03.030. Misuse Risk

Stable ID: `T07.A03.S030`

- `Line 1` If a record can be easily misused, label it and route it carefully. (`Sec. 7.03.030.L001`; stable ID `T07.A03.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 7.03.030.L002`; stable ID `T07.A03.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 7.03.030.L003`; stable ID `T07.A03.S030.L003`)
