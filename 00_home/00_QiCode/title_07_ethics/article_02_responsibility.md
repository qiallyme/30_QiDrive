## Article 2. Responsibility

Stable ID: `T07.A02`

### Sec. 7.02.010. Owned Action

Stable ID: `T07.A02.S010`

- `Line 1` Responsibility begins with what I can truthfully own, repair, change, or refuse. (`Sec. 7.02.010.L001`; stable ID `T07.A02.S010.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 7.02.010.L002`; stable ID `T07.A02.S010.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 7.02.010.L003`; stable ID `T07.A02.S010.L003`)

### Sec. 7.02.020. False Burden

Stable ID: `T07.A02.S020`

- `Line 1` I am not responsible for carrying another person's entire system when consent, capacity, or safety is absent. (`Sec. 7.02.020.L001`; stable ID `T07.A02.S020.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 7.02.020.L002`; stable ID `T07.A02.S020.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 7.02.020.L003`; stable ID `T07.A02.S020.L003`)

### Sec. 7.02.030. Evidence

Stable ID: `T07.A02.S030`

- `Line 1` Responsibility claims should be grounded in evidence and context, not projection alone. (`Sec. 7.02.030.L001`; stable ID `T07.A02.S030.L001`)
- `Line 2` Apply this provision with truth, care, consent, and proportion. (`Sec. 7.02.030.L002`; stable ID `T07.A02.S030.L002`)
- `Line 3` If the provision is used in a project, link back to the record, output, or decision that applies it. (`Sec. 7.02.030.L003`; stable ID `T07.A02.S030.L003`)
