# Install This Package in Obsidian

## Existing Vault

1. Extract the ZIP file.
2. Copy the enclosed **`QiCode`** folder into the root of your vault.
3. Open [[00 - QiCode Home|QiCode Home]].

## New Standalone Vault

Open the folder containing `QiCode` as an Obsidian vault, then open `QiCode/00 - QiCode Home.md`.

## Notes

- No community plugin is required.
- The package uses standard Markdown, YAML properties, and Obsidian wikilinks.
- No `.obsidian` settings are included, so this package will not overwrite your vault configuration.
- When a `QiCode` folder already exists, compare or back it up before replacing files.
