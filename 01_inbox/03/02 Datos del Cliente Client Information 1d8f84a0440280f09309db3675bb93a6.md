# 02. Datos del Cliente \Client Information

Keywords: Gathering essential details about the client for the audit.
Section Name: Audit Prep
Task Description: "Preparación de la Auditoría - Datos del Cliente:
Primero, es importante reunir detalles esenciales sobre el cliente para garantizar que la auditoría sea completa y precisa. Esto incluye información sobre la estructura del cliente, su situación financiera y cualquier otro dato relevante que pueda impactar el proceso de auditoría.

Tareas a Realizar:
1. Reunir información básica del cliente (nombre, dirección, tipo de negocio).
2. Obtener registros financieros relevantes del cliente.
3. Revisar la información recopilada para asegurar su precisión.
4. Identificar cualquier área que requiera atención especial durante la auditoría."
