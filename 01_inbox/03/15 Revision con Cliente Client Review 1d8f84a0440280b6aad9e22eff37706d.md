# 15. Revision con Cliente \Client Review

Keywords: Reviewing the compiled audit package with the client.
Section Name: Audit Prep
Task Description: Revisión con Cliente - Preparación de Auditoría
"Primero, queremos revisar el paquete de auditoría compilado con el cliente para asegurarnos de que toda la información es precisa y está completa. Esto nos ayudará a identificar cualquier área que necesite atención antes de la auditoría final."

Elementos de Acción:
1. Revisar el paquete de auditoría compilado.
2. Identificar cualquier discrepancia o área que necesite aclaración.
3. Documentar las observaciones realizadas durante la revisión.
4. Programar una reunión de seguimiento con el cliente para discutir los hallazgos.
