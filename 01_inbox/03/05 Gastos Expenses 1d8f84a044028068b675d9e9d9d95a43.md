# 05. Gastos \Expenses

Keywords: Gathering documentation related to the business's expenses.
Section Name: Audit Prep
Task Description: "Preparación de Auditoría - Gastos" "Primero, es necesario reunir toda la documentación relacionada con los gastos del negocio y asegurarse de que todo esté en orden para la auditoría. Esto incluye revisar recibos, facturas y cualquier otro documento que respalde los gastos reportados."

Elementos de acción:
1. Recopilar toda la documentación de gastos.
2. Revisar los recibos y facturas para asegurarse de que estén completos.
3. Verificar que los gastos estén debidamente clasificados.
4. Documentar cualquier discrepancia o falta de documentación.
