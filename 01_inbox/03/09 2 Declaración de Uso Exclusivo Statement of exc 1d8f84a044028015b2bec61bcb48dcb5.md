# 09.2 Declaración de Uso Exclusivo \Statement of exclusive use

Keywords: Documenting that the home office space is used exclusively for business.
Section Name: Audit Prep
Task Description: "Preparación de Auditoría - Declaración de Uso Exclusivo"
"Primero, queremos documentar que el espacio de la oficina en casa se utiliza exclusivamente para fines comerciales. Esto es importante para asegurar el cumplimiento con las regulaciones fiscales y para maximizar las deducciones disponibles."

Elementos de acción:
1. Revisar la documentación relacionada con el uso del espacio de la oficina.
2. Asegurarse de que el espacio cumple con los requisitos para su uso exclusivo como oficina.
3. Documentar el uso exclusivo y los detalles relevantes.
4. Preparar cualquier evidencia necesaria para respaldar la declaración.
