# Follow Up

Keywords: follow up, next steps, nurture, reminder, second contact
Section Name: Lead
Task Description: Para la sección "Lead", es importante realizar un seguimiento con el cliente potencial para asegurar que se sientan valorados y para fomentar una relación más sólida. Esto implica recordarles sobre su interés y ofrecer asistencia adicional si es necesario.

Acciones a realizar:
1. Revisar la información del cliente potencial.
2. Enviar un correo electrónico o mensaje de seguimiento.
3. Preguntar si tienen alguna pregunta o necesitan más información.
4. Programar una llamada si es necesario para discutir más a fondo.
