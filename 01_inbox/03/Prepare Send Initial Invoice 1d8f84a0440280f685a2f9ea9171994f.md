# Prepare/Send Initial Invoice

Keywords: billing, first bill, initial invoice, payment request
Section Name: Onboarding
Task Description: Para la tarea "Preparar/Enviar Factura Inicial", primero se debe reunir toda la información necesaria para la factura, asegurándose de que todos los servicios prestados estén correctamente documentados. Esto incluye verificar las tarifas acordadas y la información del cliente.

Elementos de acción:
1. Reunir la información de los servicios prestados.
2. Verificar las tarifas y detalles acordados con el cliente.
3. Redactar la factura inicial.
4. Revisar la factura para asegurar su precisión.
5. Enviar la factura al cliente.
