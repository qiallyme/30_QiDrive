# 03.3 Comunicaciones Estatales \State Correspondence

Keywords: Collecting any communication from state tax authorities.
Section Name: Audit Prep
Task Description: Comunicaciones Estatales - Preparación de Auditoría
"Primero, es importante recopilar todas las comunicaciones de las autoridades fiscales estatales y asegurarse de que estén organizadas. Esto facilitará la revisión durante el proceso de auditoría."

Acciones a Realizar:
1. Recopilar todas las comunicaciones relevantes de las autoridades fiscales estatales.
2. Organizar las comunicaciones por fecha y tipo.
3. Revisar las comunicaciones para identificar cualquier área de preocupación.
4. Documentar las comunicaciones y cualquier acción requerida en respuesta.
