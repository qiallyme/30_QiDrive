# 03.5 Registros estatales anuales \Annual state filings

Keywords: Obtaining annual reports or filings made with the state.
Section Name: Audit Prep
Task Description: Preparación de Auditoría - Registros Estatales Anuales
Primero, queremos revisar los informes anuales o presentaciones realizadas ante el estado para garantizar que toda la información esté completa y cumpla con los requisitos legales. Es fundamental identificar cualquier discrepancia y asegurarse de que se tomen las medidas correctivas necesarias.

Tareas:
1. Obtener los informes anuales relevantes.
2. Revisar los registros y buscar discrepancias.
3. Documentar cualquier discrepancia encontrada.
4. Desarrollar o enumerar acciones necesarias para corregir las discrepancias.
