# 06.3 Forma W-2, W-3, 941, 940 \forms

Keywords: Collecting employer payroll tax forms.
Section Name: Audit Prep
Task Description: ## Preparación de Auditoría - Formularios de Impuestos

"En esta sección, se deben recopilar y revisar los formularios de impuestos de nómina del empleador para garantizar que estén completos y precisos. Es esencial verificar que todos los formularios requeridos estén disponibles y que se hayan presentado a tiempo."

### Tareas:

1. Recopilar todos los formularios W-2, W-3, 941 y 940.
2. Revisar cada formulario para detectar errores o información faltante.
3. Asegurarse de que todos los formularios se hayan presentado dentro de los plazos establecidos.
4. Documentar cualquier discrepancia o problema encontrado durante la revisión.
5. Preparar un informe sobre el estado de los formularios de impuestos para la auditoría.
