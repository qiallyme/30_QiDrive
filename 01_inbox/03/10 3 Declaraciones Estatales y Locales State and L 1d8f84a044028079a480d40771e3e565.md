# 10.3 Declaraciones Estatales y Locales \State and Local Returns

Keywords: Gathering copies of state and local tax returns.
Section Name: Audit Prep
Task Description: Declaraciones Estatales y Locales - Preparación de Auditoría
"Primero, queremos recopilar las copias de las declaraciones de impuestos estatales y locales para garantizar que toda la información esté en orden antes de la auditoría. Esto incluye revisar las declaraciones para identificar cualquier discrepancia o problema potencial."

Acciones:
1. Recopilar copias de las declaraciones de impuestos estatales y locales.
2. Revisar las declaraciones en busca de discrepancias.
3. Documentar cualquier problema encontrado.
4. Preparar un resumen de las declaraciones para la auditoría.
