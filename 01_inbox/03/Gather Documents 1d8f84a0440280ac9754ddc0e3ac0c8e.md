# Gather Documents

Keywords: client information, document collection, records, tax documents
Section Name: Tax Prep
Task Description: Para la sección "Preparación de Impuestos", el objetivo es reunir todos los documentos necesarios para la preparación de impuestos del cliente. Esto incluye información clave del cliente y documentos fiscales que son esenciales para completar la declaración de impuestos de manera precisa y eficiente.

Acciones a realizar:
1. Recopilar la información del cliente.
2. Obtener todos los documentos fiscales necesarios.
3. Revisar la documentación para asegurarse de que esté completa.
4. Organizar los documentos en un formato accesible para la preparación de impuestos.
