# QiApps

## Overview

QiApps holds standalone user interfaces, web applications, and utility sites built on top of the QiOS platform.

## Responsibilities

- Host primary user applications (QiLife, QiJourney, QiAccess).
- Keep application-level configuration and deployment assets isolated.
- Provide compiled static outputs for browser viewing.

## Folder Structure & Table of Contents

### Core Applications

- [QiLife](QiLife/): Lifedesk personal data dashboard and visual tracker.
- [QiJourney](QiJourney/): Dynamic chronology visualization and event logs dashboard.
- [QiAccess Start](qiaccess_start/): System cockpit portal launcher interface.
- [Static Site Compiler](site/): The static compiled HTML output for DNA documentation.
