# Ask QiLife: UI Flow
1. **Console Input**: Cody types a natural language query in the Ask QiLife console.
2. **Answer Panel**: The system renders a Markdown-formatted response from the AI.
3. **Citations & References**: The UI displays interactive cards representing database records cited by the AI.
4. **Follow-up Actions**: Quick-actions allow Cody to save the AI response as a Note, Knowledge Item, or new Action.
