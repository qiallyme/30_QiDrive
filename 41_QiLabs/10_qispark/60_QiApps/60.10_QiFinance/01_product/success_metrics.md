# Success Metrics

## MVP Success

- User can enter 20 transactions without confusion.
- User can split mixed transactions correctly.
- Dashboard balances match ledger totals.
- User can explain where money went for a selected period.

## Alpha Success

- A friend can use the app without Cody explaining every screen.
- Demo mode can be reset safely.
- Data can be exported.
- No Cody-only assumptions remain in code paths.
