# Prompt Guidelines

## Good Prompt Pattern

Give the AI:

1. Current goal
2. Relevant files
3. Constraints
4. Acceptance criteria
5. What not to change

## Example

Build the split transaction form. Use the existing account model. Do not change schema. Validate debits equal credits before save. Keep UI plain and mobile-friendly.
