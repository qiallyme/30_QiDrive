# Storage & Sync Settings
QiLife manages its own data footprint instead of relying on manual user folders.

## Planned Tables
- `app_settings`
- `storage_locations`
- `sync_targets`
- `sync_runs`
- `system_health_checks`

The system must regularly verify DB mode, sync mode, backup status, and storage health.
