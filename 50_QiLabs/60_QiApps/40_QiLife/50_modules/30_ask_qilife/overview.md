# Ask QiLife: Overview
The **Ask QiLife** module is the natural language query interface for the entire LifeOps ledger. It aligns with the **Personal LifeDesk** model by serving as the desk's virtual supervisor, helping Cody retrieve info, trace logs, and resolve daily chaos without hunting through tables.
