# QiFinance Obsidian Docs Pack

This zip contains an Obsidian-ready documentation directory for QiFinance.

## How to Use

1. Unzip into your Obsidian vault or project folder.
2. Open the `docs/` folder in Obsidian.
3. Start with `docs/00_doctrine/index.doctrine.md`.
4. Use `docs/08_ai/context_pack.md` when handing the project to AI coding tools.

## Structure

- `00_doctrine` — purpose, principles, ADRs
- `01_product` — users, problems, goals, roadmap
- `02_infrastructure` — environments, tenancy, auth strategy, storage
- `03_domain` — business rules, workflows, entities
- `04_architecture` — system design and module map
- `05_data` — schema, RLS, migrations, import/export
- `06_api` — endpoints, contracts, integrations
- `07_ui` — routes, navigation, components
- `08_ai` — AI context and coding guidance
- `09_operations` — setup, deployment, testing, recovery
- `10_history` — changelog, decisions, retrospectives
- `templates` — reusable doc templates
