# Agent Instructions

## Rules for Coding Agents

- Read doctrine before changing architecture.
- Do not introduce auth until requested.
- Preserve workspace_id fields.
- Do not replace double-entry with single-category budgeting.
- Add or update docs when changing architecture or data model.
- Keep changes modular.
