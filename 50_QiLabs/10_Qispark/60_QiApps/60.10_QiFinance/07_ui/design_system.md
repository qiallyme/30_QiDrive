# Design System

## Style Direction

Clean, direct, low-clutter, high-contrast.

## UI Priorities

- Fast entry
- Clear totals
- Strong empty states
- Obvious validation errors
- Mobile-friendly layouts

## Avoid

- Cute budget app fluff
- Dense accounting screens as default
- Hidden required fields
