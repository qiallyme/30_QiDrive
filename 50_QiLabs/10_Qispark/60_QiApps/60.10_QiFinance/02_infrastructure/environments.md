# Environments

## LOCAL_DEV

For local development and fast iteration.

## TEST_USER_DEMO

For friend testing with seeded demo data.

## PROD_SINGLE_USER

Private production for Cody or first real usage.

## PROD_MULTI_USER_LATER

Future authenticated product environment.

## Required Pattern

Use environment variables for project URLs, anon keys, feature flags, and demo mode controls.
