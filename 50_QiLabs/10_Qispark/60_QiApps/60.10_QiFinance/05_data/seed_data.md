# Seed Data

## Purpose

Seed data should allow fast testing and friend demos.

## Include

- Demo workspace
- Demo user
- Checking account
- Cash account
- Credit card
- Vehicle expense accounts
- Gig income accounts
- Sample split transactions

## Rule

Demo data must be resettable without touching real data.
