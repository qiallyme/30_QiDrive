# Submitted applications

Create one folder only after an application is submitted:

`YYYY-MM-DD_company_role/`

Copy [[_submission_template]] into it and add:

- `job_posting.md` or a PDF/screenshot of the posting
- Exact résumé and cover letter submitted
- `application_answers.md` for material free-text answers
- Confirmation email, screenshot, PDF, or confirmation number
- Recruiter/contact information

The tracker remains the quick overview; this folder is the supporting evidence.
