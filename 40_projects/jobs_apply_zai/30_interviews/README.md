# Interviews

Create one note per stage: `YYYY-MM-DD_company_stage.md`.

Record the address or meeting link, transportation plan, interviewer names, questions and answers, documents requested, schedule/pay discussed, next steps, and any language assistance needed. Never record a call without everyone’s lawful consent.
