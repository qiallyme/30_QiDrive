# Submitted applications

Create one folder after each application is submitted:

`YYYY-MM-DD_company_role/`

Copy [[_submission_template]] into it and add:

- The job posting as Markdown, PDF, or screenshot
- The exact résumé and cover note submitted
- Important written application answers
- Confirmation email, screenshot, PDF, or confirmation number
- Recruiter/contact information

The tracker is the quick overview; this folder preserves the proof and details.
