# Quick Start

## 1. Put this folder in your vault

Copy `MS-IL_Reentry_Obsidian_Workspace` into the desired project area inside your Obsidian vault.

## 2. Open the canvas

Open [[MS-IL Reentry Mission.canvas]] for the campaign map.

## 3. Open the dashboard

Use [[00_HOME/Dashboard]] as the daily starting point.

## 4. Work from tasks, not from memory

Use [[06_TASKS/Master Task List]] as the only authoritative task list.

## 5. Record every contact

After each email, text, call, or application, update [[07_TRACKING/Contact Log]].

## 6. Preserve clean versions

- Ready-to-send copy belongs in `02_OUTREACH`.
- Rough wording belongs in `09_DRAFTS`.
- Superseded material belongs in `99_ARCHIVE`.

## Suggested Obsidian features

- Pin [[00_HOME/Dashboard]]
- Add [[06_TASKS/Master Task List]] to bookmarks
- Use backlinks to track all notes connected to Elvis, Matt, and Tugso
- Add the Templates core plugin and point it to `08_TEMPLATES`
- Add Tasks or Dataview later only if useful; this workspace works without plugins
