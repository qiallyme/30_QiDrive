# System Failures Summary

## Current Summary

The record shows a support-system failure pattern: the household and care arrangement relied heavily on Cody while formal care, family backup, pay systems, transportation systems, and administrative systems did not reliably absorb the burden.

## Main Threads

- Comfort Keepers / Humana role and pay mismatch.
- Lack of backup support.
- Care coordination pushed onto one person.
- Transportation and phone systems failing during care crisis.
- Need to build Family OS / care dashboard / evidence ledger because the informal system was not working.
