# How To Use This Vault

## Fast Path

1. Start with [[01_dashboard.care_record]].
2. Use [[00_timeline_master]] for the dated spine.
3. Use [[00_worklog_master]] for labor and care categories.
4. Use [[01_internal_exhibit_index]] for claims and exhibit buckets.
5. Use [[2026-06-30_source_inventory_review]] to see what was searched, what was merged, and what still needs raw proof.

## Evidence Rule

Keep claims, summaries, and raw proof separate. Summaries should name where the proof lives; raw evidence should stay in original form or in [[40_projects/Lisa_Care_Record_2026/99_EVIDENCE_RAW/99_ARCHIVE_RAW]].
